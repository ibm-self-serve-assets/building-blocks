# Zero-Copy Lakehouse

Enable seamless querying across databases, warehouses, and cloud object stores without data duplication using IBM watsonx.data.

## What's Included

### Assets

- **[Setup Lakehouse](./assets/setup-lakehouse/)**: Python automation script for watsonx.data setup including COS, S3, and DB2 integration
  - Automates watsonx.data resource setup
  - Registers IBM COS and AWS S3 buckets
  - Connects DB2 databases
  - Configures Presto engine catalogs
