# Data Enrichment & Quality Capability

**Core Capability**: Enrichment
**IBM Products**: watsonx.data Intelligence
**Product Components**: Governance and Catalog; Data Quality; Data Lineage; Metadata enrichment; Intelligent search; Data product sharing

## Overview

Text-to-SQL, NLQ-to-SQL, governed RAG/Q&A, Open RAG. Improves trust and usability of enterprise data through cataloging, governance, quality controls, lineage, metadata enrichment, and discovery.

## Key Features

- **Governance and Catalog**: Centralized data governance and cataloging
- **Data Quality**: Automated quality assessment and validation
- **Data Lineage**: Track data flow and transformations
- **Metadata Enrichment**: Enhance data with business context
- **Intelligent Search**: AI-powered data discovery
- **Data Product Sharing**: Enable data collaboration across teams
- **Text-to-SQL**: Natural language to SQL query generation
- **NLQ-to-SQL**: Natural language query processing
- **Governed RAG/Q&A**: Secure retrieval-augmented generation
- **Open RAG**: Flexible RAG implementations

## Use Cases

- Enterprise data cataloging and discovery
- Data quality monitoring and improvement
- Compliance and governance tracking
- Metadata management and enrichment
- Natural language data access
- Secure AI-powered Q&A over enterprise data
