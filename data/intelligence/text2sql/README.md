# Text2SQL

**Core Capability**: Intelligence
**IBM Products**: watsonx.data Intelligence
**Product Components**: Text2SQL

## Overview

Convert natural language questions to SQL queries. This capability enables users to query databases using natural language, making data access more intuitive and accessible.

## Key Features

- **Natural Language Processing**: Convert questions to SQL automatically
- **Metadata-Enriched Queries**: Leverage database schema and metadata
- **Enterprise Q&A**: Enable business users to query data without SQL knowledge
- **Configurable Pipelines**: Customize query generation and execution
- **Multi-Database Support**: Works with various SQL databases

## Assets

This building block contains:
- `assets/` - Text2SQL implementation, applications, demos, and examples
- `bob-modes/` - Bob AI assistant modes for Text2SQL automation

## Quick Start

1. Navigate to the asset directory
2. Follow the setup instructions in the asset README
3. Configure your database connection
4. Start querying with natural language

## Use Cases

- **Business Intelligence**: Enable non-technical users to query data
- **Data Exploration**: Quick insights without writing SQL
- **Report Generation**: Natural language report queries
- **Data Analysis**: Ad-hoc analysis with conversational queries
