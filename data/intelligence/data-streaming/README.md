# Data Streaming Capability

**Core Capability**: Enrichment
**IBM Products**: Confluent
**Product Components**: IBM integration with Apache Kafka topics, connectors, and Apache Flink

## Overview

Supports real-time event ingestion, streaming pipelines, and stream processing for operational and analytical use cases. This capability enables continuous data flow and real-time processing across your data infrastructure.

## Key Features

- **Real-time Event Ingestion**: Capture and process events as they occur
- **Streaming Pipelines**: Build continuous data processing workflows
- **Apache Kafka Integration**: Industry-standard event streaming platform
- **Apache Flink Support**: Advanced stream processing capabilities
- **Operational Use Cases**: Real-time monitoring and alerting
- **Analytical Use Cases**: Stream analytics and real-time insights
