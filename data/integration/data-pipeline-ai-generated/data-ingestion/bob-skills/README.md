# confluent-solution-architect

Brief instructions for installing and using the `confluent-solution-architect` skill package.

## Install

1. Download `confluent-solution-architect.zip`.
2. Unzip the file.
3. Copy or move the extracted `confluent-solution-architect` folder into your `.bob` directory.

Expected location:

- Windows: `C:\Users\<your-user>\.bob\confluent-solution-architect`
- macOS/Linux: `~/.bob/confluent-solution-architect`

## Use

After the folder is placed in `.bob`, restart Bob if it is already open.

Then activate or ask for the skill by name:

- `Use the confluent-solution-architect skill`
- `Design a Confluent Cloud streaming solution for my use case`
- `Generate Terraform, Flink SQL, and Python producers for this architecture`

## What this skill helps with

- Confluent Cloud solution architecture
- Terraform-based infrastructure setup
- Apache Flink SQL pipeline design
- Python producer examples
- Business-focused streaming solution scaffolding
