# Real-Time Guardrails

Enforce safety boundaries and operational constraints to keep your AI applications within desired behavior in production.

---

## What's Inside

### [Assets](assets/)
A Dash-based demo application showing how to use the watsonx governance SDK to implement real-time guardrails for generative AI models — content safety, bias detection, RAG quality metrics, and more. Guardrails can be applied to both AI inputs and outputs.
