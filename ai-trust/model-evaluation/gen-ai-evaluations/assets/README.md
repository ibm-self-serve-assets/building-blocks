# Gen AI Evaluations — Assets

Technical assets for evaluating generative AI applications. Choose the folder that matches your needs:

| Folder | Use When | What It Provides |
|--------|----------|-----------------|
| [evaluation-scripts/](evaluation-scripts/) | You want to **quickly evaluate** a GenAI application | 5 standalone Python scripts covering RAG quality, content safety, LLM-as-judge, readability, and deployment readiness |
| [evaluation-sdk/](evaluation-sdk/) | You want to **integrate evaluation into your application** | Importable Python package (`wx_gov_prompt_eval`) with PromptTemplateEvaluator class, SLM and LLM-as-Judge modes, OpenScale monitoring |
