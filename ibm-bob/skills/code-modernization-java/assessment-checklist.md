- [ ] Map all source files by language
- [ ] Identify framework versions
- [ ] Document build system (Maven, npm, pip, etc.)
- [ ] List deployment architecture
- [ ] Map CI/CD pipelines
- [ ] Identify database technologies
- [ ] Document external integrations

## Dependency Analysis
- [ ] List all dependencies with versions

- [ ] Measure test coverage percentage
- [ ] Identify God classes (>500 LOC)

- [ ] Detect duplicate code
- [ ] Find dead/unused code
- [ ] Check cyclomatic complexity

## Integration Points

- [ ] Document database connections
- [ ] Map file system dependencies

- [ ] Document message queues/events

- [ ] Document deployment process
- [ ] Identify compliance requirements
- [ ] Check security requirements

- [ ] Security vulnerabilities
- [ ] Performance bottlenecks
- [ ] Scalability limitations
- [ ] Data migration complexity
